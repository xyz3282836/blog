                                                                                          еңЁ OOP зЁӢеәҸи®ҫи®ЎдёӯпјҢеҪ“жҲ‘д»¬е®ҡд№үдёҖдёӘ class зҡ„ж—¶еҖҷпјҢеҸҜд»Ҙд»ҺжҹҗдёӘзҺ°жңүзҡ„ class з»§жүҝпјҢж–°зҡ„ class з§°дёәеӯҗзұ» (Subclass) пјҢиҖҢиў«з»§жүҝзҡ„ class з§°дёәеҹәзұ»гҖҒзҲ¶зұ»жҲ–и¶…зұ» (Base classгҖҒSuper class) гҖӮ

                                                                                          жҜ”еҰӮпјҢжҲ‘д»¬е·Із»Ҹзј–еҶҷдәҶдёҖдёӘеҗҚдёә Animal зҡ„ classпјҢжңүдёҖдёӘ run() ж–№жі•еҸҜд»ҘзӣҙжҺҘжү“еҚ°:

                                                                                          class Animal(object):
    def run(self):
        print('Animal is running...')

                                                                                          еҪ“жҲ‘д»¬йңҖиҰҒзј–еҶҷ Dog е’Ң Cat зұ»ж—¶пјҢе°ұеҸҜд»ҘзӣҙжҺҘд»Һ Animal зұ»з»§жүҝ:

                                                                                          class Dog(Animal):
    pass

class Cat(Animal):
    pass

                                                                                          еҜ№дәҺ Dog жқҘиҜҙпјҢAnimal е°ұжҳҜе®ғзҡ„зҲ¶зұ»пјҢеҜ№дәҺ Animal жқҘиҜҙпјҢDog е°ұжҳҜе®ғзҡ„еӯҗзұ»гҖӮCat е’Ң Dog зұ»дјјгҖӮ

                                                                                          з»§жүҝжңүд»Җд№ҲеҘҪеӨ„пјҹжңҖеӨ§зҡ„еҘҪеӨ„жҳҜеӯҗзұ»иҺ·еҫ—дәҶзҲ¶зұ»зҡ„е…ЁйғЁеҠҹиғҪгҖӮз”ұдәҺ Animial е®һзҺ°дәҶ run() ж–№жі•пјҢеӣ жӯӨпјҢDog е’Ң Cat дҪңдёәе®ғзҡ„еӯҗзұ»пјҢд»Җд№ҲдәӢд№ҹжІЎе№ІпјҢе°ұиҮӘеҠЁжӢҘжңүдәҶ run() ж–№жі•:

                                                                                          dog = Dog()
dog.run()

cat = Cat()
cat.run()

                                                                                          иҝҗиЎҢз»“жһңеҰӮдёӢ:

                                                                                          Animal is running...
Animal is running...

                                                                                          еҪ“з„¶пјҢд№ҹеҸҜд»ҘеҜ№еӯҗзұ»еўһеҠ дёҖдәӣж–№жі•пјҢжҜ”еҰӮ Dog зұ»:

                                                                                          class Dog(Animal):

    def run(self):
        print('Dog is running...')

    def eat(self):
        print('Eating meat...')

                                                                                          з»§жүҝзҡ„з¬¬дәҢдёӘеҘҪеӨ„йңҖиҰҒжҲ‘д»¬еҜ№д»Јз ҒеҒҡдёҖзӮ№ж”№иҝӣгҖӮжӮЁзңӢеҲ°дәҶпјҢж— и®әжҳҜ Dog иҝҳжҳҜ CatпјҢе®ғд»¬ run() зҡ„ж—¶еҖҷпјҢжҳҫзӨәзҡ„йғҪжҳҜ Animal is running...пјҢз¬ҰеҗҲйҖ»иҫ‘зҡ„еҒҡжі•жҳҜеҲҶеҲ«жҳҫзӨә Dog is running... е’Ң Cat is running...пјҢеӣ жӯӨпјҢеҜ№ Dog е’Ң Cat зұ»ж”№иҝӣеҰӮдёӢ:

                                                                                          class Dog(Animal):

    def run(self):
        print('Dog is running...')

class Cat(Animal):

    def run(self):
        print('Cat is running...')

                                                                                          еҶҚж¬ЎиҝҗиЎҢпјҢз»“жһңеҰӮдёӢ:

                                                                                          Dog is running...
Cat is running...

                                                                                          еҪ“еӯҗзұ»е’ҢзҲ¶зұ»йғҪеӯҳеңЁзӣёеҗҢзҡ„ run() ж–№жі•ж—¶пјҢжҲ‘д»¬иҜҙпјҢеӯҗзұ»зҡ„ run() иҰҶзӣ–дәҶзҲ¶зұ»зҡ„ run()пјҢеңЁд»Јз ҒиҝҗиЎҢзҡ„ж—¶еҖҷпјҢжҖ»жҳҜдјҡи°ғз”Ёеӯҗзұ»зҡ„ run()гҖӮиҝҷж ·пјҢжҲ‘д»¬е°ұиҺ·еҫ—дәҶз»§жүҝзҡ„еҸҰдёҖдёӘеҘҪеӨ„: еӨҡжҖҒгҖӮ

                                                                                          иҰҒзҗҶи§Јд»Җд№ҲжҳҜеӨҡжҖҒпјҢжҲ‘д»¬йҰ–е…ҲиҰҒеҜ№ж•°жҚ®зұ»еһӢеҶҚдҪңдёҖзӮ№иҜҙжҳҺгҖӮеҪ“жҲ‘д»¬е®ҡд№үдёҖдёӘ class зҡ„ж—¶еҖҷпјҢжҲ‘д»¬е®һйҷ…дёҠе°ұе®ҡд№үдәҶдёҖз§Қж•°жҚ®зұ»еһӢгҖӮжҲ‘д»¬е®ҡд№үзҡ„ж•°жҚ®зұ»еһӢе’Ң Python иҮӘеёҰзҡ„ж•°жҚ®зұ»еһӢпјҢжҜ”еҰӮ strгҖҒlistгҖҒdict жІЎд»Җд№ҲдёӨж ·:

                                                                                          a = list() # aжҳҜlistзұ»еһӢ
b = Animal() # bжҳҜAnimalзұ»еһӢ
c = Dog() # cжҳҜDogзұ»еһӢ

                                                                                          еҲӨж–ӯдёҖдёӘеҸҳйҮҸжҳҜеҗҰжҳҜжҹҗдёӘзұ»еһӢеҸҜд»Ҙз”Ё isinstance() еҲӨж–ӯ:

                                                                                          >>> isinstance(a, list)
True
>>> isinstance(b, Animal)
True
>>> isinstance(c, Dog)
True

                                                                                          зңӢжқҘ aгҖҒbгҖҒc зЎ®е®һеҜ№еә”зқҖ listгҖҒAnimalгҖҒDog иҝҷ 3 з§Қзұ»еһӢгҖӮ

                                                                                          дҪҶжҳҜзӯүзӯүпјҢиҜ•иҜ•:

                                                                                          >>> isinstance(c, Animal)
True

                                                                                          зңӢжқҘ c дёҚд»…д»…жҳҜ DogпјҢc иҝҳжҳҜ Animal!

                                                                                          дёҚиҝҮд»”з»ҶжғіжғіпјҢиҝҷжҳҜжңүйҒ“зҗҶзҡ„пјҢеӣ дёә Dog жҳҜд»Һ Animal з»§жүҝдёӢжқҘзҡ„пјҢеҪ“жҲ‘д»¬еҲӣе»әдәҶдёҖдёӘ Dog зҡ„е®һдҫӢ c ж—¶пјҢжҲ‘д»¬и®Өдёә c зҡ„ж•°жҚ®зұ»еһӢжҳҜ Dog жІЎй”ҷпјҢдҪҶ c еҗҢж—¶д№ҹжҳҜ Animal д№ҹжІЎй”ҷпјҢDog жң¬жқҘе°ұжҳҜ Animal зҡ„дёҖз§Қ!

                                                                                          жүҖд»ҘпјҢеңЁз»§жүҝе…ізі»дёӯпјҢеҰӮжһңдёҖдёӘе®һдҫӢзҡ„ж•°жҚ®зұ»еһӢжҳҜжҹҗдёӘеӯҗзұ»пјҢйӮЈе®ғзҡ„ж•°жҚ®зұ»еһӢд№ҹеҸҜд»Ҙиў«зңӢеҒҡжҳҜзҲ¶зұ»гҖӮдҪҶжҳҜпјҢеҸҚиҝҮжқҘе°ұдёҚиЎҢ:

                                                                                          >>> b = Animal()
>>> isinstance(b, Dog)
False

                                                                                          Dog еҸҜд»ҘзңӢжҲҗ AnimalпјҢдҪҶ Animal дёҚеҸҜд»ҘзңӢжҲҗ DogгҖӮ

                                                                                          иҰҒзҗҶи§ЈеӨҡжҖҒзҡ„еҘҪеӨ„пјҢжҲ‘д»¬иҝҳйңҖиҰҒеҶҚзј–еҶҷдёҖдёӘеҮҪж•°пјҢиҝҷдёӘеҮҪж•°жҺҘеҸ—дёҖдёӘ Animal зұ»еһӢзҡ„еҸҳйҮҸ:

                                                                                          def run_twice(animal):
    animal.run()
    animal.run()

                                                                                          еҪ“жҲ‘д»¬дј е…Ҙ Animal зҡ„е®һдҫӢж—¶пјҢrun_twice() е°ұжү“еҚ°еҮә:

                                                                                          >>> run_twice(Animal())
Animal is running...
Animal is running...

                                                                                          еҪ“жҲ‘д»¬дј е…Ҙ Dog зҡ„е®һдҫӢж—¶пјҢrun_twice() е°ұжү“еҚ°еҮә:

                                                                                          >>> run_twice(Dog())
Dog is running...
Dog is running...

                                                                                          еҪ“жҲ‘д»¬дј е…Ҙ Cat зҡ„е®һдҫӢж—¶пјҢrun_twice() е°ұжү“еҚ°еҮә:

                                                                                          >>> run_twice(Cat())
Cat is running...
Cat is running...

                                                                                          зңӢдёҠеҺ»жІЎе•Ҙж„ҸжҖқпјҢдҪҶжҳҜд»”з»ҶжғіжғіпјҢзҺ°еңЁпјҢеҰӮжһңжҲ‘д»¬еҶҚе®ҡд№үдёҖдёӘ Tortoise зұ»еһӢпјҢд№ҹд»Һ Animal жҙҫз”ҹ:

                                                                                          class Tortoise(Animal):
    def run(self):
        print('Tortoise is running slowly...')

                                                                                          еҪ“жҲ‘д»¬и°ғз”Ё run_twice() ж—¶пјҢдј е…Ҙ Tortoise зҡ„е®һдҫӢ:

                                                                                          >>> run_twice(Tortoise())
Tortoise is running slowly...
Tortoise is running slowly...

                                                                                          жӮЁдјҡеҸ‘зҺ°пјҢж–°еўһдёҖдёӘ Animal зҡ„еӯҗзұ»пјҢдёҚеҝ…еҜ№ run_twice() еҒҡд»»дҪ•дҝ®ж”№пјҢе®һйҷ…дёҠпјҢд»»дҪ•дҫқиө– Animal дҪңдёәеҸӮж•°зҡ„еҮҪж•°жҲ–иҖ…ж–№жі•йғҪеҸҜд»ҘдёҚеҠ дҝ®ж”№ең°жӯЈеёёиҝҗиЎҢпјҢеҺҹеӣ е°ұеңЁдәҺеӨҡжҖҒгҖӮ

                                                                                          еӨҡжҖҒзҡ„еҘҪеӨ„е°ұжҳҜпјҢеҪ“жҲ‘д»¬йңҖиҰҒдј е…Ҙ DogгҖҒCatгҖҒTortoiseвҖҰвҖҰж—¶пјҢжҲ‘д»¬еҸӘйңҖиҰҒжҺҘж”¶ Animal зұ»еһӢе°ұеҸҜд»ҘдәҶпјҢеӣ дёә DogгҖҒCatгҖҒTortoiseвҖҰвҖҰйғҪжҳҜ Animal зұ»еһӢпјҢз„¶еҗҺпјҢжҢүз…§ Animal зұ»еһӢиҝӣиЎҢж“ҚдҪңеҚіеҸҜгҖӮз”ұдәҺ Animal зұ»еһӢжңү run() ж–№жі•пјҢеӣ жӯӨпјҢдј е…Ҙзҡ„д»»ж„Ҹзұ»еһӢпјҢеҸӘиҰҒжҳҜ Animal зұ»жҲ–иҖ…еӯҗзұ»пјҢе°ұдјҡиҮӘеҠЁи°ғз”Ёе®һйҷ…зұ»еһӢзҡ„ run() ж–№жі•пјҢиҝҷе°ұжҳҜеӨҡжҖҒзҡ„ж„ҸжҖқ:

                                                                                          еҜ№дәҺдёҖдёӘеҸҳйҮҸпјҢжҲ‘д»¬еҸӘйңҖиҰҒзҹҘйҒ“е®ғжҳҜ Animal зұ»еһӢпјҢж— йңҖзЎ®еҲҮең°зҹҘйҒ“е®ғзҡ„еӯҗзұ»еһӢпјҢе°ұеҸҜд»Ҙж”ҫеҝғең°и°ғз”Ё run() ж–№жі•пјҢиҖҢе…·дҪ“и°ғз”Ёзҡ„ run() ж–№жі•жҳҜдҪңз”ЁеңЁ AnimalгҖҒDogгҖҒCat иҝҳжҳҜ Tortoise еҜ№иұЎдёҠпјҢз”ұиҝҗиЎҢж—¶иҜҘеҜ№иұЎзҡ„зЎ®еҲҮзұ»еһӢеҶіе®ҡпјҢиҝҷе°ұжҳҜеӨҡжҖҒзңҹжӯЈзҡ„еЁҒеҠӣ: и°ғз”Ёж–№еҸӘз®Ўи°ғз”ЁпјҢдёҚз®Ўз»ҶиҠӮпјҢиҖҢеҪ“жҲ‘д»¬ж–°еўһдёҖз§Қ Animal зҡ„еӯҗзұ»ж—¶пјҢеҸӘиҰҒзЎ®дҝқ run() ж–№жі•зј–еҶҷжӯЈзЎ®пјҢдёҚз”Ёз®ЎеҺҹжқҘзҡ„д»Јз ҒжҳҜеҰӮдҪ•и°ғз”Ёзҡ„гҖӮиҝҷе°ұжҳҜи‘—еҗҚзҡ„вҖңејҖй—ӯвҖқеҺҹеҲҷ:

                                                                                          еҜ№жү©еұ•ејҖж”ҫ: е…Ғи®ёж–°еўһ Animal еӯҗзұ»пјӣ

                                                                                          еҜ№дҝ®ж”№е°Ғй—ӯ: дёҚйңҖиҰҒдҝ®ж”№дҫқиө– Animal зұ»еһӢзҡ„ run_twice() зӯүеҮҪж•°гҖӮ

                                                                                          з»§жүҝиҝҳеҸҜд»ҘдёҖзә§дёҖзә§ең°з»§жүҝдёӢжқҘпјҢе°ұеҘҪжҜ”д»ҺзҲ·зҲ·еҲ°зҲёзҲёгҖҒеҶҚеҲ°е„ҝеӯҗиҝҷж ·зҡ„е…ізі»гҖӮиҖҢд»»дҪ•зұ»пјҢжңҖз»ҲйғҪеҸҜд»ҘиҝҪжәҜеҲ°ж №зұ» objectпјҢиҝҷдәӣз»§жүҝе…ізі»зңӢдёҠеҺ»е°ұеғҸдёҖйў—еҖ’зқҖзҡ„ж ‘гҖӮжҜ”еҰӮеҰӮдёӢзҡ„з»§жүҝж ‘:

    pass

                                                                                          зҺ°еңЁпјҢжҲ‘д»¬иҰҒз»ҷеҠЁзү©еҶҚеҠ дёҠ Runnable е’Ң Flyable зҡ„еҠҹиғҪпјҢеҸӘйңҖиҰҒе…Ҳе®ҡд№үеҘҪ Runnable е’Ң Flyable зҡ„зұ»:

                                                                                          class Runnable(object):
    def run(self):
        print('Running...')

class Flyable(object):
    def fly(self):
        print('Flying...')

                                                                                          class Dog(Mammal, Runnable):

                                                                                          class Bat(Mammal, Flyable):
    pass

                                                                                          # MixIn

                                                                                          еңЁи®ҫи®Ўзұ»зҡ„з»§жүҝе…ізі»ж—¶пјҢйҖҡеёёпјҢдё»зәҝйғҪжҳҜеҚ•дёҖз»§жүҝдёӢжқҘзҡ„пјҢдҫӢеҰӮпјҢOstrich з»§жүҝиҮӘ BirdгҖӮдҪҶжҳҜпјҢеҰӮжһңйңҖиҰҒвҖңж··е…ҘвҖқйўқеӨ–зҡ„еҠҹиғҪпјҢйҖҡиҝҮеӨҡйҮҚз»§жүҝе°ұеҸҜд»Ҙе®һзҺ°пјҢжҜ”еҰӮпјҢи®© Ostrich йҷӨдәҶз»§жүҝиҮӘ Bird еӨ–пјҢеҶҚеҗҢж—¶з»§жүҝ RunnableгҖӮиҝҷз§Қи®ҫи®ЎйҖҡеёёз§°д№Ӣдёә MixInгҖӮ

                                                                                          дёәдәҶжӣҙеҘҪең°зңӢеҮәз»§жүҝе…ізі»пјҢжҲ‘д»¬жҠҠ Runnable е’Ң Flyable ж”№дёә RunnableMixIn е’Ң FlyableMixInгҖӮзұ»дјјзҡ„пјҢжӮЁиҝҳеҸҜд»Ҙе®ҡд№үеҮәиӮүйЈҹеҠЁзү©CarnivorousMixIn е’ҢжӨҚйЈҹеҠЁзү© HerbivoresMixInпјҢи®©жҹҗдёӘеҠЁзү©еҗҢж—¶жӢҘжңүеҘҪеҮ дёӘ MixIn:

                                                                                          class Dog(Mammal, RunnableMixIn, CarnivorousMixIn):
    pass

                                                                                          MixIn зҡ„зӣ®зҡ„е°ұжҳҜз»ҷдёҖдёӘзұ»еўһеҠ еӨҡдёӘеҠҹиғҪпјҢиҝҷж ·пјҢеңЁи®ҫи®Ўзұ»зҡ„ж—¶еҖҷпјҢжҲ‘д»¬дјҳе…ҲиҖғиҷ‘йҖҡиҝҮеӨҡйҮҚз»§жүҝжқҘз»„еҗҲеӨҡдёӘ MixIn зҡ„еҠҹиғҪпјҢиҖҢдёҚжҳҜи®ҫи®ЎеӨҡеұӮж¬Ўзҡ„еӨҚжқӮзҡ„з»§жүҝе…ізі»гҖӮ

                                                                                          Python иҮӘеёҰзҡ„еҫҲеӨҡеә“д№ҹдҪҝз”ЁдәҶ MixInгҖӮдёҫдёӘдҫӢеӯҗпјҢPython иҮӘеёҰдәҶ TCPServer е’Ң UDPServer иҝҷдёӨзұ»зҪ‘з»ңжңҚеҠЎпјҢиҖҢиҰҒеҗҢж—¶жңҚеҠЎеӨҡдёӘз”ЁжҲ·е°ұеҝ…йЎ»дҪҝз”ЁеӨҡиҝӣзЁӢжҲ–еӨҡзәҝзЁӢжЁЎеһӢпјҢиҝҷдёӨз§ҚжЁЎеһӢз”ұ ForkingMixIn е’Ң ThreadingMixIn жҸҗдҫӣгҖӮйҖҡиҝҮз»„еҗҲпјҢжҲ‘д»¬е°ұеҸҜд»ҘеҲӣйҖ еҮәеҗҲйҖӮзҡ„жңҚеҠЎжқҘгҖӮ

                                                                                          жҜ”еҰӮпјҢзј–еҶҷдёҖдёӘеӨҡиҝӣзЁӢжЁЎејҸзҡ„ TCP жңҚеҠЎпјҢе®ҡд№үеҰӮдёӢ:

                                                                                          class MyTCPServer(TCPServer, ForkingMixIn):
    pass

                                                                                          зј–еҶҷдёҖдёӘеӨҡзәҝзЁӢжЁЎејҸзҡ„ UDP жңҚеҠЎпјҢе®ҡд№үеҰӮдёӢ:

                                                                                          class MyUDPServer(UDPServer, ThreadingMixIn):
    pass

                                                                                          еҰӮжһңжӮЁжү“з®—жҗһдёҖдёӘжӣҙе…Ҳиҝӣзҡ„еҚҸзЁӢжЁЎеһӢпјҢеҸҜд»Ҙзј–еҶҷдёҖдёӘ CoroutineMixIn:

                                                                                          class MyTCPServer(TCPServer, CoroutineMixIn):
    pass

                                                                                          иҝҷж ·дёҖжқҘпјҢжҲ‘д»¬дёҚйңҖиҰҒеӨҚжқӮиҖҢеәһеӨ§зҡ„з»§жүҝй“ҫпјҢеҸӘиҰҒйҖүжӢ©з»„еҗҲдёҚеҗҢзҡ„зұ»зҡ„еҠҹиғҪпјҢе°ұеҸҜд»Ҙеҝ«йҖҹжһ„йҖ еҮәжүҖйңҖзҡ„еӯҗзұ»гҖӮ

                                                                                          # е°Ҹз»“

                                                                                          з»§жүҝеҸҜд»ҘжҠҠзҲ¶зұ»зҡ„жүҖжңүеҠҹиғҪйғҪзӣҙжҺҘжӢҝиҝҮжқҘпјҢиҝҷж ·е°ұдёҚеҝ…йҮҚйӣ¶еҒҡиө·пјҢеӯҗзұ»еҸӘйңҖиҰҒж–°еўһиҮӘе·ұзү№жңүзҡ„ж–№жі•пјҢд№ҹеҸҜд»ҘжҠҠзҲ¶зұ»дёҚйҖӮеҗҲзҡ„ж–№жі•иҰҶзӣ–йҮҚеҶҷгҖӮ

                                                                                          еҠЁжҖҒиҜӯиЁҖзҡ„йёӯеӯҗзұ»еһӢзү№зӮ№еҶіе®ҡдәҶз»§жүҝдёҚеғҸйқҷжҖҒиҜӯиЁҖйӮЈж ·жҳҜеҝ…йЎ»зҡ„гҖӮ

                                                                                          з”ұдәҺ Python е…Ғи®ёдҪҝз”ЁеӨҡйҮҚз»§жүҝпјҢеӣ жӯӨпјҢMixIn е°ұжҳҜдёҖз§Қеёёи§Ғзҡ„и®ҫи®ЎгҖӮ
